Project Overview

Project 1 - E-Commerce Dataset
Data in csv file, cleaning, manipulation, and visualization performed in Excel using Formulas, Pivot Tables and Charts. The analysis aims to identify the trends in sales performance and show how shipping fees influence the order transactions.
Project 2 - Financial Services Dataset
Data in csv file (49K rows) cleaning and exploration performed in PostgreSQL and visualization with Excel Charts. Analysis aims to identify the funding sources available and status.
Project 3 - Digital Marketing Dataset
Data in csv format (2.4K rows), cleaning and exploration with PostgreSQL, and visualization with Excel Charts. The analysis aims to identify the potential factors of transaction disputes leading to loss of revenue, and provide insights on preventing or minimizing the likelihood of its occurrence.
Project 4 - E-Commerce Dataset
Data in csv file (99.4K rows), cleaning, exploration, and visualization with Power Query and Power BI. The analysis aims to identify the trends in delivery performance to optimize delivery times.
Project 5 - YouTube Dataset
Data in Excel file (369K rows), cleaning, exploration, and visualization with Power Query and PBI. The analysis aims to demonstrate the performance of categories and products to improve the quality of content and increase audience engagement.
Project 6 - AI/ML Dev Dataset
Data in Excel file (4.5k rows), cleaning in Excel, exploration and visualization with Power Query and PBI. The analysis aims to minimize the potential reasons for lost sales and identify the factors driving the upsell to increase revenue.
